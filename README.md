# 🏛️ Senat Pribadi

Decision Making Assistant berbasis AI dengan pendekatan multi-perspektif

---

# 📌 Deskripsi

Senat Pribadi adalah aplikasi web sederhana yang membantu menganalisis suatu masalah dari 5 sudut pandang berbeda:

- 🧠 Logika
- ⚖️ Etika
- 🔍 Skeptis
- ❤️ Empati
- 🛠️ Pragmatis

Tujuannya adalah mensimulasikan "diskusi internal" sebelum mengambil keputusan.

---

# ⚙️ Fitur

- Input masalah bebas
- Analisis multi-role secara otomatis
- Animasi "sidang" (opening & closing)
- Typing effect (AI response simulation)
- Mode gelap / terang
- Ringkasan akhir (kesimpulan)
